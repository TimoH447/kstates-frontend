About K.States

K.States is a web application for visualizing state lattices of knots. You can input a knot diagram in PD notation and select a fixed segment to generate the associated Kauffman state lattice.

The site returns a visualization of the lattice as well as relevant metadata, such as:

  • Number of Kauffman states
  • Number of lattice layers
  • Minimum and maximum marker positions
  • Sequence of transpositions from min to max state

This tool is designed for knot theorists, researchers, and enthusiasts who work with diagrammatic invariants or are exploring the combinatorics of state models.

This project was created by Timo Hauser.
